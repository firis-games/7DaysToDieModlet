7Days to Die a19 Modlet
===

7DtDプレイ用に調整した追加したModlet

---
## 対応バージョン
- 7Days to Die a19.6

## 概要
- 一部アイテムのスタック数を増加
- シャム缶/チェーンソー/オーガのパークによるレシピ解放を追加
- 殲滅系トレーダークエストを削除
- ワールド自動販売機で商品の売却ができるように変更
- カジノトークンのバンドルアイテムを追加
- 抗生物質効果のある食べ物のサバイバルスープ缶を追加
- 乗り物のインベントリを拡張
- ロボット発明家のパーク効果を上方修正（アクティブ数と有効距離）
- アーチェリーのパーク効果を上方修正（リロード速度アップ）
- バンドル系アイテムのレシピを追加
- 切り株の蜂蜜ドロップ数とドロップ率を増加
- 日本語訳を一部変更
